/send

Usage:

/SEND [-W] [-Ttype] [-w[world]] [-n] text

Sends text to a world.

Options:

-wworld
sends text to world.
-Ttype
sends text to all connected worlds with a type that matches the pattern type.
-W
sends text to all connected worlds.
-n
send text without an end-of-line marker (CR LF).

If none of the -w, -W, or -T options are given, text is sent to the current world.

The /send command never executes a SEND hook.

See: send().
